## HandyPurse

Raises stack limits for currencies:
- Scrap: 9999
- Black Coin: 999
- Crystals (BlackBlood and Glitter): 999

## Config (easy mode)

HandyPurse uses the built-in BepInEx config file format (`.cfg`, INI-like).

This repo tracks a default config at:
- `HandyPurse/Config/io.github.fantastic-jam.raidersofblackveil.mods.handypurse.cfg`

Deploy it with:
- `deploy.bat HandyPurse`

After first launch with the mod enabled, edit:
- `BepInEx/config/io.github.fantastic-jam.raidersofblackveil.mods.handypurse.cfg`

Available settings:
- `Limits.ScrapCap`
- `Limits.BlackCoinCap`
- `Limits.CrystalCap`

Restart the game after changing values.
