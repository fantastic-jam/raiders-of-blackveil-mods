# GhoulagUrskov

Adds Urskov as a fully-featured playable character with unique abilities, stats, and progression mechanics.

## Requirements

- [Raiders of Blackveil](https://store.steampowered.com/app/2018990/Raiders_of_Blackveil/)
- [BepInEx 5](https://github.com/BepInEx/BepInEx)

## Installation

1. Ensure BepInEx 5 is installed in your Raiders of Blackveil game directory.
2. Download the latest release ZIP from the [releases page](https://github.com/fantastic-jam/raiders-of-blackveil-mods/releases?q=GhoulagUrskov).
3. Extract the contents to your game folder. Files should merge into `BepInEx/plugins/`.
4. Launch the game — the mod loads automatically.
