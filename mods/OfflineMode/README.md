# OfflineMode

Allows you to play Raiders of Blackveil without an internet connection. Your game state is saved locally, and up to 20 backups are kept in case something goes wrong.

When you return online, the game will prompt you to choose between your cloud save and your local save.

**Note:** you must have played the game at least once before Offline Mode becomes available.

---

## Requirements

- [Raiders of Blackveil](https://store.steampowered.com/app/2246780/Raiders_of_Blackveil/)
- [BepInEx 5](https://github.com/BepInEx/BepInEx/releases)

---

## Installation

### 1. Install BepInEx

Skip this step if BepInEx is already installed.

1. Download **BepInEx 5** from the [BepInEx releases page](https://github.com/BepInEx/BepInEx/releases) — pick the `BepInEx_win_x64` build.
2. Extract the contents into your game folder (the one containing `RoB.exe`).
3. Launch the game once and close it — BepInEx will initialize its folder structure.

### 2. Install the mod

1. Download `OfflineMode-x.x.x.zip` from the [releases page](https://github.com/fantastic-jam/raiders-of-blackveil-mods/releases?q=OfflineMode).
2. Extract the ZIP into your game's `BepInEx` folder.
3. Launch the game.
