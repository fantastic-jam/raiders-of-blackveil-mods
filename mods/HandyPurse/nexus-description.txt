[size=5]HandyPurse[/size]

Raises the stack limits for all currencies so you stop hitting the cap mid-run.

[list]
[*]Scrap : 9 999[/*]
[*]Black Coin: 999[/*]
[*]Crystals (BlackBlood & Glitter): 999[/*]
[/list]

All caps are configurable.

[heading]Installation[/heading]
1. Install BepInEx (Skip this step if BepInEx is already installed)
[list]
[*]Download [b]BepInEx 5[/b] from the [url=https://github.com/BepInEx/BepInEx/releases]BepInEx releases page[/url] — pick the BepInEx_win_x64 build[/*]
[*]Extract the contents into your game folder (the one containing RoB.exe).[/*]
[*]Launch the game once and close it — BepInEx will initialize its folder structure.[/*]
[/list]
2. Install the mod
[list]
[*]Download. [/*]
[*]Extract the ZIP into your game's BepInEx folder.[/*]
[*]Launch the game.[/*]
[/list]
[line]
[heading]Configuration[/heading]
After the first launch, a config file is created at:
[code]BepInEx/config/io.github.fantastic-jam.raidersofblackveil.mods.handypurse.cfg[/code]

Open it with any text editor:

[code]
[Limits]
## Max stack for Scrap.
# Setting type: Int32
# Default value: 9999
ScrapCap = 9999

## Max stack for Black Coin.
# Setting type: Int32
# Default value: 999
BlackCoinCap = 999

## Max stack for Crystals (BlackBlood and Glitter).
# Setting type: Int32
# Default value: 999
CrystalCap = 999[/code]

Set any value to whatever cap you want. Restart the game after saving.
