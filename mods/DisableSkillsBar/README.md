# DisableSkillsBar

Prevents the skills bar from reacting to accidental hover or clicks during combat.

By default, mousing over the skills bar while fighting unlocks your cursor and disables your attack input — a death sentence mid-fight. This mod blocks that behavior, including direct ability upgrades from clicking, unless you are deliberately holding **Alt**.

No configuration required.

---

## Requirements

- [Raiders of Blackveil](https://store.steampowered.com/app/2246780/Raiders_of_Blackveil/)
- [BepInEx 5](https://github.com/BepInEx/BepInEx/releases)

---

## Installation

### 1. Install BepInEx

Skip this step if BepInEx is already installed.

1. Download **BepInEx 5** from the [BepInEx releases page](https://github.com/BepInEx/BepInEx/releases) — pick the `BepInEx_win_x64` build.
2. Extract the contents into your game folder (the one containing `RoB.exe`).
3. Launch the game once and close it — BepInEx will initialize its folder structure.

### 2. Install the mod

1. Download `DisableSkillsBar-x.x.x.zip` from the [releases page](https://github.com/fantastic-jam/raiders-of-blackveil-mods/releases?q=DisableSkillsBar).
2. Extract the ZIP directly into your game folder.
3. Launch the game.
