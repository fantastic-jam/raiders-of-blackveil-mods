[size=5]CheatManager[/size]

Enables the game's built-in developer cheat hotkeys in release builds.

The game ships with a full set of cheat hotkeys (heal, kill enemies, add currency, XP, etc.) that are locked out in release builds. This mod unlocks them. Hold [b]Alt[/b] in-game to display a hotkey reference overlay.

[heading]Hotkeys[/heading]
[list]
[*][b]H[/b] — +25% health[/*]
[*][b]U[/b] — Heal all players to full[/*]
[*][b]L[/b] — Kill all enemies[/*]
[*][b]Shift+L[/b] — Trigger level exit[/*]
[*][b]Shift+K[/b] — Restart same level[/*]
[*][b]M[/b] — +200 Black Coins[/*]
[*][b]Shift+M[/b] — +2500 Scrap[/*]
[*][b]N[/b] — +50 XP for all players[/*]
[*][b]9[/b] — Set all-players-dead flag[/*]
[*][b]Shift+B[/b] — Force vending machine[/*]
[*][b]F4 / F11 / F12[/b] — Damage player slots 0 / 1 / 2[/*]
[/list]

Hotkeys are only active when no UI windows are open.
[line]
[heading]ModManager[/heading]
This mod registers with [url=https://www.nexusmods.com/raidersofblackveil/mods/12?tab=files]ModManager[/url] as a [b]Cheat[/b]. When ModManager is installed, an [b]Allow Cheats[/b] toggle appears in the host setup screen. If the host sets it to [b]No[/b], CheatManager is disabled for that session and the session name will show a [b](cheats)[/b] suffix when it's active.
