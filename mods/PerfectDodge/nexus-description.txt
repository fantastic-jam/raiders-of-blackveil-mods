[size=5]PerfectDodge[/size]

Adds a [b]just-dodge[/b] mechanic: if you get hit within a short window right after pressing dash, the hit is fully blocked and your dash charge is instantly refunded. A [i]dodged[/i] label appears overhead to confirm it.
The timing window is configurable.
[line]
[heading]WMF[/heading]
This mod registers with [url=https://www.nexusmods.com/raidersofblackveil/mods/12?tab=files]WMF[/url] as a [b]Mod[/b]. When WMF is installed, an [b]Allow Mods[/b] toggle appears in the host setup screen. If the host sets it to [b]No[/b], PerfectDodge is disabled for that session and the session name will show a [b](modded)[/b] suffix when it's active.

[heading]Installation[/heading]

1. Install BepInEx (Skip this step if BepInEx is already installed)
[list]
[*]Download [b]BepInEx 5[/b] from the [url=https://github.com/BepInEx/BepInEx/releases]BepInEx releases page[/url] — pick the BepInEx_win_x64 build[/*]
[*]Extract the contents into your game folder (the one containing RoB.exe).[/*]
[*]Launch the game once and close it — BepInEx will initialize its folder structure.[/*]
[/list]
2. Install the mod
[list]
[*]Download.[/*]
[*]Extract the ZIP into your game's BepInEx folder.[/*]
[*]Launch the game.[/*]
[/list]
[line]
[heading]Configuration[/heading]
After the first launch, a config file is created at:
[code]BepInEx/config/io.github.fantastic-jam.raidersofblackveil.mods.perfectdodge.cfg[/code]

[list]
[*][b]PerfectDodgeWindowSeconds[/b] — Default: [b]0.3[/b] | Range: 0.01 – 1.0 | How many seconds after pressing dash the just-dodge window stays open.[/*]
[/list]

Restart the game after saving.
