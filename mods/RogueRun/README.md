# RogueRun

> The run that never happened. The one they still talk about.

They say my grandfather walked into the Meat Factory with nothing but the clothes on his back. No smuggler on the Liberator that day — no way to send anything home. So he armed himself with whatever Blackveil left lying around and cleared every room on the way down. Walked out alive. Left it all inside where it belonged. His own kit was waiting right where he'd stashed it. I've heard the story a hundred times. It gets a little longer every year.

---

## Highlights

- **Went in empty-handed** — permanent loadout locked away safe while the run kit builds itself room by room from nothing
- **Every door, every room, no shortcuts** — the whole gauntlet, top to bottom, the way the old-timers say it was meant to be run
- **Loot dropped on top of loot** — gear chests stacked over normal rewards, because in that run, the Factory was generous
- **Bosses handed over their secrets** — perks straight from every major kill, each fight shaping what you'd become by the end
- **No junk cluttering the satchel** — stat flasks where the souvenirs and recipes would've been; numbers climbed clean the whole way through
- **Killed the BiomeBoss and walked** — original gear untouched, the run itself gone, nothing left but the memory

## WMF

RogueRun registers with [WMF](https://github.com/fantastic-jam/raiders-of-blackveil-mods/releases?q=WildguardModFramework) as a **Game Mode**. Select it in the host setup screen before starting a session. All clients must have RogueRun installed and enabled in WMF to join.

## Installation

### 1. Install BepInEx

Skip this step if BepInEx is already installed.

1. Download **BepInEx 5** from the [BepInEx releases page](https://github.com/BepInEx/BepInEx/releases) — pick the `BepInEx_win_x64` build.
2. Extract the contents into your game folder (the one containing `RoB.exe`).
3. Launch the game once and close it — BepInEx will initialize its folder structure.

### 2. Install the mod

1. Download `RogueRun-x.x.x.zip` from the [releases page](https://github.com/fantastic-jam/raiders-of-blackveil-mods/releases?q=RogueRun).
2. Extract the ZIP into your game's `BepInEx` folder.
3. Launch the game.
