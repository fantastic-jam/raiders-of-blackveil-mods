# PlayerNameFix

Fixes the `<N/A>` display name shown when your Steam profile is private. At login the game cannot read your Steam name from the backend, so it stores `<N/A>` permanently on your account. This mod overrides that name locally with your actual Steam persona name before it is broadcast to other players.

---

## Requirements

- [Raiders of Blackveil](https://store.steampowered.com/app/2246780/Raiders_of_Blackveil/)
- [BepInEx 5](https://github.com/BepInEx/BepInEx/releases)

---

## Installation

### 1. Install BepInEx

Skip this step if BepInEx is already installed.

1. Download **BepInEx 5** from the [BepInEx releases page](https://github.com/BepInEx/BepInEx/releases) — pick the `BepInEx_win_x64` build.
2. Extract the contents into your game folder (the one containing `RoB.exe`).
3. Launch the game once and close it — BepInEx will initialize its folder structure.

### 2. Install the mod

1. Download `PlayerNameFix-x.x.x.zip` from the [releases page](https://github.com/fantastic-jam/raiders-of-blackveil-mods/releases?q=PlayerNameFix).
2. Extract the ZIP into your game's `BepInEx` folder.
3. Launch the game.
