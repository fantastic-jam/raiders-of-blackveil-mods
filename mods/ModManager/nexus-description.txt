[size=5]ModManager[/size]

[color=#ff6600][b]⚠ Deprecated.[/b] ModManager has been superseded by [url=https://github.com/fantastic-jam/raiders-of-blackveil-mods/releases?q=WildguardModFramework]WMF[/url]. Please migrate — ModManager will be removed in a future update.[/color]

ModManager is the previous version of the mod management runtime for Raiders of Blackveil. It is no longer actively developed. All features have been moved to [b]WMF[/b], which also adds a network channel API for mod-to-mod communication.
[line]
[heading]Migration[/heading]
[list]
[*]Uninstall ModManager ([font=Courier New]BepInEx/plugins/fantastic-jam-ModManager/[/font])[/*]
[*]Install [url=https://github.com/fantastic-jam/raiders-of-blackveil-mods/releases?q=WildguardModFramework]WMF[/url] from the releases page[/*]
[*]If your mod declares [font=Courier New][BepInDependency(ModManagerMod.Id)][/font], update it to [font=Courier New][BepInDependency(WMFMod.Id)][/font][/*]
[/list]
Mods that use duck typing (no DLL reference) require no code changes.
